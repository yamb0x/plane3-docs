| Format | Import | Export | Notes |
|--------|--------|--------|-------|
| GLB | Yes | Yes | Binary glTF. Primary format used internally. |
| glTF | Yes | Yes | JSON-based glTF with separate `.bin` and texture files. |
| OBJ | Yes | Yes | Wavefront OBJ with MTL material support. Multi-object preserved. |
| FBX | Yes | No | Autodesk FBX. Materials normalized to PBR on import. |
| STL | Yes | Yes | Geometry only, no materials or textures. |
