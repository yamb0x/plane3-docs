| Shortcut | Action |
|----------|--------|
| `Double-click` canvas | Open the generate menu |
| `Q` / `E` | Rotate camera view 90 degrees |
| `Scroll wheel` | Zoom in or out |
| `Middle mouse` / `Shift + Left click` | Pan the canvas |
| `C` | Activate sticky note tool |
| `P` | Activate sketch tool |
| `Delete` / `Backspace` | Delete selected objects |
| `Ctrl + Z` / `Cmd + Z` | Undo |
| `Ctrl + Shift + Z` / `Cmd + Shift + Z` | Redo |
| `/` | Open cursor chat |
| `Escape` | Deselect or close active panel |
