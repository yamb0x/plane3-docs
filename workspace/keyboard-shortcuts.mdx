## Canvas shortcuts

These shortcuts control the viewport and object interaction:

| Shortcut | Action |
|----------|--------|
| `Q` / `E` | Rotate camera view 90 degrees counterclockwise / clockwise |
| `Scroll wheel` | Zoom in or out |
| `Middle mouse` / `Shift + Left click` | Pan the canvas |
| `Double-click` canvas | Open the generate menu |
| `Delete` / `Backspace` | Delete selected objects |
| `Escape` | Deselect all or close the active panel |

## Annotation shortcuts

These shortcuts activate annotation tools:

| Shortcut | Action |
|----------|--------|
| `C` | Activate the sticky note tool. Click the canvas to place a note. |
| `P` | Activate the sketch tool. Click and drag to draw on the canvas. |

## General shortcuts

These shortcuts work across the entire application:

| Shortcut | Action |
|----------|--------|
| `Ctrl + Z` / `Cmd + Z` | Undo the last action |
| `Ctrl + Shift + Z` / `Cmd + Shift + Z` | Redo the last undone action |
| `/` | Open cursor chat for inline communication with collaborators |
| `Escape` | Close the active panel or deselect objects |
